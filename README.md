# Tkinterplus
Tkinterplus is a python UI library that adds more widgets to Tkinter

## Installation
Install the module with pip:
```bat
pip3 install tkinterplus
```
Update existing installation: `pip3 install tkinterplus --upgrade`

## Documentation
